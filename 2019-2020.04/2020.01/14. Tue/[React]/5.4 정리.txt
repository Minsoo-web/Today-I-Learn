컴포넌트 내부에서 DOM에 직접 접근해야 할 때는 ref를 사용합니다 
먼저 ref를 활용하지 않고도 원하는 기능을 구현할 수 있는지 반드시 고려한 후에 활용하세요 

서로 다른 컴포넌트끼리 데이터를 교류할 때 ref를 사용한다면 잘못 사용된 것 ! 

앱 규모가 커지면 마치 스파게티처럼 구조가 꼬여버려서 유지 보수가 불가능해짐 

컴포넌트끼리 데이터를 교류할 때는 언제나 데이터를 부모 <-> 자식 흐름으로 교류해야 합니다. 
나중에 리덕스 혹은 context API 를 사용하여 효율적으로 교류하는 방법을 배울 것입니다. 

함수형 컴포넌트에서는 useRef라는 Hook 함수를 사용합니다
